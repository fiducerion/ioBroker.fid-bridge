# ioBroker.fid-bridge

Teil der Fiducerion-Adapter-Familie fuer ioBroker.

## Installation (privates Repo via Personal Access Token)

```bash
# Erste Installation
sudo -u iobroker iobroker url "https://${GH_PAT}@github.com/fiducerion/ioBroker.fid-bridge/tarball/v0.12.5"

# Update auf neueste main-Branch
sudo -u iobroker iobroker url "https://${GH_PAT}@github.com/fiducerion/ioBroker.fid-bridge/tarball/main"
```
